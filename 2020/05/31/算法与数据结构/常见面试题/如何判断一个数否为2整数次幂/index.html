如何判断一个数否为2整数次幂

阅读量: 今晚打老虎 2020-05-31 20:43:04

1. logn 版本

1.1. 原理

创建一个中间变量，初始值是 1，开始循环，每次循环都让 temp 和目标数对比

如果相等，说明目标数是 2 的整数次幂

如果不相等，则让 temp增大 1 倍，继续循环进行比较

当 temp 大于目标整数时，说明目标数不是 2 的整数次幂

举个例子

目标数 19：

1x2 = 2;

2x2 = 4;

4x2 = 8;

8x2 = 16;

16x2 = 32;

由于 32 大于 19，所以 19 不是 2 的整数次幂

1.2. 代码实现-1

还是 logn

public class 一个数是否是2的整数次幂 {
	public static void main(String[] args) {
		System.out.println(isPowerOf2(32));
		System.out.println(isPowerOf2(19));
	}

	static boolean isPowerOf2(int num) {
		int temp = 1;
		while (temp <= num) {
			// 如果相等,说明时
			if (temp == num) {
				return true;
			}
			// 否则temp 翻倍
			temp = temp * 2;
		}
		return false;
	}
}

1.3. 代码实现-2

还是 logn

乘以的操作可以改成向左位移，位移的性能比乘法算法高

	static boolean isPowerOf2V2(int num) {
		int temp = 1;
		while (temp <= num) {
			// 如果相等,说明时
			if (temp == num) {
				return true;
			}
			// 否则temp 翻倍
			temp = temp << 1;
		}
		return false;
	}

2. o1 版本

2.1. 思路

  1. 把 2 的整数次幂转换成二进制，有这样的共同点

    如果一个整数是 2 的整数次幂，那么当它转化成二进制是，只有最高位是 1，其他位都是 0

  1. 把这些 2 的整数次幂各自减 1，再转化成二进制

    一旦减 1，他的二进制数组就全部变成了 1

  2. 用原来数和它减 1 的结果进行按位与运算，也就是 n&（n-1）

    0 和 1 按位与运算的结果0，所以，凡是 2 的整数次幂和它本身减 1 的结算进行与运算，结果必然0

    反之，如果不是 2 的整数次幂，那么结果一定不是 0

2.1.1. 总结：

对于一个整数 n，只需要计算 n&（n-1）到结果是否是 0，复杂度只有 o1

2.2. 代码实现

	/**
	 * o1 复杂度 按位与运算
	 * @param num
	 * @return
	 */
	static boolean isPowerOf2V3(int num) {
		return (num & num - 1) == 0;
	}
