主要方法还是通过使用本机自带的M.2接口进行转接实现使用苹果原装无线网卡，为了避免大家多花钱走坑，所以就写了这么一个帖子。

适用机型NUC8i5BEK、NUC8i5BEH、NUC8i7BEK、NUC8i7BEH。

1. 购买转接卡

目前针对M.2接口的转接卡有两款：M.2 key M接口绿色卡、M.2 key B+M黑色款。

两款的区别是绿色的更便宜，USB蓝牙连接口为1.0mm4pin，插座离转接卡接口很近；黑色款USB蓝牙接口为1.0mm2pin，插座在网卡的后面。

这两张卡板我都用过，从效果上来说两张卡效果完全一样，唯一区别就是黑色卡版更硬朗一些，有指示灯，可以在两种M.2插槽上使用，绿色卡板只能在一种M.2插槽上使用。目前价格上绿色卡板为¥37，黑色卡板为¥55。

2. 购买连线配件

由于这两款转接卡使用的都是1.0mm端子排线输出到PC主板USB插孔的，而NUC8代内置的两个USB2.0插座为1.25mm间距4P的插座，所以并不匹配。

绿色卡版插座为1.0mm间距4P规格，黑色卡版插座为1.0mm间距2P规格。

关于连线有两种方法，一种是懒人方式一种是无焊接方式；

2.1. ¶懒人方式：

只需要购买一个1.25mm间距4P带排线插孔并和转接卡附送的连接线进行对接即可，只要记着红色的排线要跟红色的排线对接并用绝缘胶带包裹即可。

2.2. ¶无焊接方式：

如果你追求更好看的连接效果可以通过杜邦压线钳进行压制或者购买现成端子转接线。

无焊接方式也有两种，一种是购买现成的1.25mm转1.0mm的端子连接线，第二种是自己压制任意长度的端子连接线。

2.2.1. ¶购买现成的端子连接线：

压制好的1.25mm转1.0mm端子连接线，20厘米长一件10根 ¥1.98；

1.25mm间距胶壳4P插孔，一件10只 ¥0.20；

1.0mm间距胶壳4P插孔，一件10只 ¥0.56；

算上邮费一共 ¥7元。

以上配件安装方式也很简单，把两根端子线分别插入相应胶壳即可，无需焊接。

2.2.2. ¶自己压制端子连接线：

购买杜邦1.0mm/1.25mm压线钳 ¥55；

购买1.0mm间距连接排线（带胶壳插孔）4P，长度15cm ¥0.36；

购买1.25mm间距胶壳4P插孔，一件10只 ¥0.20；

购买SH 1.25MM母端子，一件50个 ¥1.1元；

算上邮费约 ¥62。

2.2.2.1. ¶以上配件到货后就可以开始压制了；

1.因为已经买了带1.0mm间距插孔的排线，所以你只需要压制连接NUC8内置USB2.0接口的端子即可；

2.修建掉一点金属线长度；

3.把线放入1.25MM母端子内，塑料部分要和母电子最后的凸起卡线槽处对其，金属线部分和母端子中间卡线槽处对其；

4.用剪刀或者细钳子把母端子末端的卡槽稍微从两边压紧一下电子线；

5.轻轻把母电子从排片上分离放入压线钳0.7的位置分别压制末端和中间处即可完成端子的压制。

把压制好所需要的两根端子线插入1.25MM间距胶壳就可以开始安装了。

如果你还不懂可以随便淘宝找一个卖杜邦压线钳的店铺就可以看到视频教程。

对于转接卡完全了解之后我们可以开始选购无线网卡。

3. 购买苹果原装无线网卡

购买完转接卡之后接下来要选购苹果原装无线网卡，一共有三款可以选择：BCM94360CDP/BCM943602CS/BCM94360CS2。

以上卡根据你的喜欢选择，因为考虑到NUC8i5内置天线只有两根，所以我选购了BCM943602CS，到货之后发现蓝牙只有在很近的距离才不卡顿，离开主机30cm之后就开始卡顿严重，而且与天线没有关系，跟WiFi也没有关系，就连绿联的USB蓝牙都比这货稳，最后找奸商退掉重新入了一张BCM94360CS2，收货后接上内置的两根天线后一切完美。

所以大家只要发现自己买的网卡蓝牙有卡顿就要马上退货或者换货。

开始安装苹果原装无线网卡：

1.确定好网卡尾部的空位所在位置，然后安装垫高螺丝，如下图所示；

2.把原装无线网卡安装到转卡上；

3.安装上螺丝固定好原装无线网卡；

4.把NUC8主机的内置天线整理出来；

5.把转接卡插到主机M.2接口里面；

6.上螺丝把转接卡固定；

7.把天线安装好；

8.插好转接卡蓝牙线；

9.最后把连接NUC8内置USB的插座的排线插入就完成最后的安装。

10.如果你想布线好看点可以把线从转接卡底部穿过。

完成以上的安装之后开机进入系统之后就可以到蓝牙和WiFi已经免驱了。
