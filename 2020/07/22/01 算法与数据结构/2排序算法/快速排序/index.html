快速排序

阅读量: 今晚打老虎 2020-07-22 09:48:22

基于冒泡排序演变而来的

快的原因是使用了分治法。

跟冒泡一样，都是属于交换排序，通过元素之间的比较和交换位置来达到排序的目的

跟冒泡不同的是：冒泡在每一轮中只把1 个元素冒泡到数列的一端，而快排则是每一轮挑选一个基准元素，并让其他比他大的元素移动到数列一遍，比他小的元素移动到数列的另一边，从而把数列拆解成了两个部分

这种思路就叫做分治法

时间复杂度

logn

基准元素的选择

即：轴

以基准元素为中心，把其他元素移动到它的左右两边

选择首元素，或者随机选择

元素的交换

把小于基准元素的都放到一边，大于基准元素的放到另一边

具体如何实现呢？两种方法

  1. 双边循环法
  2. 单边循环法

双边循环法

流程

原始数组

选定轴，设置两个指针 left 和 right，指向最左和最右两个元素

  1. 第一次循环：从 right 指针开始，让指针锁指向的元素和轴做比较
    1. 大于等于轴，right 指针往左移动
    2. 小于轴，right 停止移动，切换到 left 指针
  2. left 指针元与轴作比较
    1. 小于等于轴，left 指针往右移动
    2. 大于轴，left 停止移动
  3. 此时，两个指针都停止移动了，让 right 指针元素与 left 指针元素做交换，一次循环完成
  4. 第二次循环。。。
  5. 第三次循环。。。

示例代码

quickSort通过递归的方式，实现了分而治之的思想

partition则实现了元素的交换，让数列中的元素依据自身大小，分别交换到基准元素的左右两边

package com.d1mq.algo;

import java.util.Arrays;

/**
 *
 * @author xzh
 * @date 2020/5/18 4:57 下午
 * @since v
 *
 */
public class 快速排序 {
	public static void main(String[] args) {
		int[] arr = new int[]{4, 4, 6, 5, 3, 2, 8, 1};
		quickSort(arr, 0, arr.length - 1);
		System.out.println(Arrays.toString(arr));
	}

	static void quickSort(int[] arr, int startIndex, int endIndex) {
		// 递归结束条件 : startIndex 大于等于 endIndex时
		if (startIndex >= endIndex) {
			return;
		}
		// 得到基准元素位置
		int partition = partition(arr, startIndex, endIndex);

		quickSort(arr, startIndex, partition - 1);
		quickSort(arr, partition + 1, endIndex);
	}

	/**
	 * 双边循环法
	 * @param arr 待交换的数组
	 * @param startIndex 起始下标
	 * @param endIndex 结束下标
	 * @return
	 */
	static int partition(int[] arr, int startIndex, int endIndex) {
		// 取第一个位置(也可以随机选择位置)的元素作为基准元素
		int pivot = arr[startIndex];
		int left = startIndex;
		int right = endIndex;

		while (left != right) {
			// 控制 right 指针比较并左移
			while (left < right && arr[right] > pivot) {
				right--;
			}
			// 控制 left 指针比较并右移
			while (left < right && arr[left] <= pivot) {
				left++;
			}
			// 交换 left 和 right 指针所指向的元素
			if (left < right) {
				swap(arr, left, right);
			}
		}
		// 轴和指针重合点交换
		arr[startIndex] = arr[left];
		arr[left] = pivot;
		return left;
	}

	static void swap(int[] arr, int i, int j) {
		int temp = arr[i];
		arr[i] = arr[j];
		arr[j] = temp;
	}
}

单边循环法

双边循环法从数组的两边交替遍历元素，虽然更直观，但是代码实现相对繁琐。

单边循环则简单的多，只从数组的一边对元素进行遍历和交换。

流程

原始数组如下

开始和双边循环法类似，首先选定基准元素 pivot。同时，设置一个 mark 指针指向数列起始位置，这个 mark 指针代表小于轴的区域边界。

接下来，从 轴下一个元素位置开始遍历数组

如果遍历到的元素大于轴，就继续往后遍历

如果遍历到的元素小于轴，则要做两件事

1. 把 mark 指针右移一位，因为小于 轴的区域边界增大了 1；
2. 让最新遍历到的元素和 mark 指针所在位置的元素交换位置，因为最新遍历到的元素小于轴的区域。

首先遍历到元素 7，7>4，所以继续遍历

接下来遍历到元素是 3，3<4，所以 mark 指针向右移动 1 位

随后，让元素 3 和 mark 指针所在位置的元素交换，因为元素 3 属于小于轴的区域

按照这个思路，继续遍历，后续步骤如图所示

实例代码-1

	/**
	 * 单边循环法
	 * @param arr 带交换的数组
	 * @param startIndex 起始下标
	 * @param endIndex 结束下表
	 * @return
	 */
	static int partition1(int[] arr, int startIndex, int endIndex) {
		//取index0 位置的元素作为轴
		int pivot = arr[startIndex];
		int mark = startIndex;
		for (int i = startIndex + 1; i <= endIndex; i++) {
			if (arr[i] < pivot) {
				mark++;
				swap(arr, mark, i);
			}
		}
		arr[startIndex] = arr[mark];
		arr[mark] = pivot;
		return mark;
	}

实例代码 -2

	static void itheimaquickSort(int[] arr, int left, int right) {
		//判断,如果左边索引比右边索引大,是不合法的,直接 return结束这个方法
		if (left > right) {
			return;
		}
		//定义变量保存轴
		int base = arr[left];
		// 变量 i,指向最左边,从左往右遍历
		int i = left;
		// 变量 j,指向最左边,从右往左遍历
		int j = right;

		//该检索了,检索是一个循环操作,因为不知道循环次数,使用 while 循环
		// 当 i 和 j 相遇的时候在循环中进行检索
		while (i != j) {
			// 先由 j 从右往左检索比基数小的,如果检索到比基准数小的就停下
			// 如果检索到比基准数大的,就一直检索下去
			while (arr[j] >= base && i < j) {
				j--; // j 往左移动
			}

			// i 从左往右检索
			while (arr[i] <= base && i < j) {
				i++; // i 往右移动
			}
			swap(arr, i, j);
		}
		// 这里跳出循环了
		// 说明 i 和 j 相遇了,此时应该将 轴与 相遇位置的元素交换
		arr[left] = arr[i];
		arr[i] = base;

		// 到这里,轴就归位了,左边的数字都比他小,右边的都比他大,
		// 开始给轴的左边排序,排到轴的左边一个数
		quickSort(arr, 0, i - 1);
		// 开始给轴的左边排序,从轴的下一个数开始排
		quickSort(arr, j + 1, right);
	}
